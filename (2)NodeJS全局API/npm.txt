5.
nrm 用来管理第三方模块的下载镜像，安装nrm,win+R 输入：npm install nrm -g
nrm ls来查看所有的镜像 
终端：nrm ls
nrm use 镜像名字 来切换镜像
终端：nrm use taobao
切换返回：nrm use npm
当原来镜像不存在某个模块，或者下载速度很慢时，可以尝试切换下载服务器
